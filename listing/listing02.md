Что выведет программа? Объяснить вывод программы. Объяснить как работают defer’ы и их порядок вызовов.

```go
package main

import (
	"fmt"
)


func test() (x int) {
	defer func() {
		x++
	}()
	x = 1
	return
}


func anotherTest() int {
	var x int
	defer func() {
		x++
	}()
	x = 1
	return x
}


func main() {
	fmt.Println(test())
	fmt.Println(anotherTest())
}
```

Ответ:
```
Вывод:
2
1

defer функции исполняются после того, как возвращаемые значения установятся return,
но перед тем, как управление вернется вызывающей программе.

В defer функциях могут быть модифицированы только именованные результаты функции.
Поэтому из test возвращается 2, а в anotherTest 1.

defer функции собираются в стек и исполняются в порядке LIFO.
Этот стек реализован в виде связанного списка из структур _defer.
У каждой структуры есть поле link *_defer (указатель на следующую структуру _defer).

```
